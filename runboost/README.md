## How to install?

1. Firstly you have to download programs like:

```bash
https://code.visualstudio.com/
```

```bash
https://nodejs.org/en/download/package-manager
```

2. Clone the repo.

```bash
git clone git@github.com:bartusrm7/RunBoost.git
```

3. Went to correct repo.

```bash
cd runboost
```

```bash
npm run dev
```
